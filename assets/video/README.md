# 비디오 파일 디렉토리

이 폴더에 배경 비디오 파일을 업로드하세요.

## 필요한 비디오

- `video.mp4` - 히어로 섹션 배경 동영상

## 비디오 가이드라인

### 형식 및 코덱
- **형식**: MP4 (H.264 코덱)
- **해상도**: 1920x1080 (Full HD) 권장
- **비트레이트**: 2-5 Mbps
- **프레임레이트**: 24-30fps

### 최적화
- **파일 크기**: 10MB 이하 권장
- **길이**: 10-30초 (루프 재생)
- **압축**: HandBrake, FFmpeg 등으로 웹 최적화

### FFmpeg 최적화 명령어
```bash
ffmpeg -i input.mp4 -c:v libx264 -crf 28 -preset slow -vf scale=1920:1080 -an video.mp4
```

### 콘텐츠 권장사항
- 어두운 톤 또는 오버레이를 고려한 밝기
- 텍스트 가독성을 해치지 않는 움직임
- 루프 재생에 적합한 자연스러운 시작/끝
- 브랜드 색상(청록, 옐로)과 조화

## 대체 옵션

비디오 파일이 없는 경우:
1. 그라디언트 배경 유지 (현재 기본값)
2. 정적 이미지 사용
3. CSS 애니메이션 배경

## 성능 고려사항

- 모바일에서는 자동재생이 음소거 상태에서만 동작
- `loading="lazy"` 미지원 (비디오는 즉시 로드)
- 초기 로딩 시간 고려
