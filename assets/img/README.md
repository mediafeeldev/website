# 이미지 파일 디렉토리

이 폴더에 프로젝트 이미지 파일을 업로드하세요.

## 필요한 이미지

### 로고
- `logo.png` 또는 `logo.svg` - MEDIA FEEL 로고

### 사례 이미지
사례 데이터에 맞춰 다음과 같은 이미지를 추가하세요:
- `cases/case-001-thumb.jpg` - 썸네일
- `cases/case-001-1.jpg` - 상세 이미지 1
- `cases/case-001-2.jpg` - 상세 이미지 2
- ...

## 이미지 가이드라인

- **형식**: JPG, PNG, WebP
- **썸네일 크기**: 600x400px (권장)
- **상세 이미지**: 1200x800px 이상 (권장)
- **최적화**: 웹 최적화된 이미지 사용 (TinyPNG 등 사용)

## 임시 플레이스홀더

현재는 그라디언트 배경과 이모지 아이콘으로 플레이스홀더를 사용하고 있습니다.
실제 이미지로 교체 시 `data/*.json` 파일의 이미지 경로를 수정하세요.
